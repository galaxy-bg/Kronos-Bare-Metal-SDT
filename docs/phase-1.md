# KDX SDT Phase-1 MVP

Phase-1 provides discovery, registration, inventory storage, heartbeat tracking, and a web UI for bare-metal servers booted from a KDX Live USB.

## Included

- Agent registration API
- Agent heartbeat API
- Inventory upload API
- Server list and detail APIs
- PostgreSQL persistence
- Web dashboard and server detail views

## Deferred

- BIOS profile management
- Firmware updates
- RAID configuration
- OS deployment
- Redfish power control
