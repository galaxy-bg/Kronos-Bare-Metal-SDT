# Changelog

## 1.1.0 - 2026-06-27

- Added KDX agent version reporting during registration and heartbeat.
- Added backend storage for reported agent version/build metadata.
- Added UI and deployment report visibility for agent version.
- Improved HPE iLO license and health discovery for validated systems.
- Introduced the vendor adapter architecture foundation for HPE-first, multi-vendor expansion.

## 1.0.0 - 2026-06-24

- Initial KDX SDT live agent registration, heartbeat, inventory upload, and control-plane dashboard.
- Added HPE iLO management actions for credential validation, iLO user creation, management network settings, and license installation.
- Added deployment report export with optional credential and NIC MAC address fields.
